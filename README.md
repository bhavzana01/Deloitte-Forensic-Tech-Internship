
# Deloitte Forensic Technology Virtual Internship

This repository contains my work from the **Deloitte Forensic Technology Virtual Internship (Forage)**.
The project focuses on analyzing **factory telemetry data** and **employee compensation equality scores** to uncover insights into operational efficiency and gender pay fairness.

---

## Contents

- `daikibo-telemetry-data.json` → raw machine downtime data
- `daiko-dashboard.png` → Tableau dashboard visualization
- `Equality Table (Edited).xlsx` → classified equality scores (Fair, Unfair, Highly Discriminative)

---

## Key Insights

- Identified factories with the highest machine downtime.
- Highlighted job roles with potential pay disparities.
- Classified equality scores into actionable categories.
